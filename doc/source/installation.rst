Installation
============

To install ``ixmp4`` you may use the python package index with ``pip`` 
or install directly from source.

with ``pip``
------------

.. code:: bash

    pip install ixmp4

from source
-----------

.. code:: bash

    git clone git@github.com:iiasa/ixmp4.git
    cd ixmp4/
    poetry install [--with dev,docs,server]
